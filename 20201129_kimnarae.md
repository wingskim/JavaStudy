Java파일 컴파일(=빌드)과 실행
```
1. .java 파일 작성
2. cmd에서 컴파일 명령어 [javac java파일명] 입력 ⇒ class파일 생성됨 ex) javac test.java
3. cmd에서 [java class파일명] 입력 ex) java test
```
